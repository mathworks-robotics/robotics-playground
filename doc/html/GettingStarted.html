Getting Started with Robotics Playground

Description

This toolbox provides a set of Simulink virtual environments to aid in robotics educations. Users can interact directly with robots via actuator and sensor blocks to learn to program robots for driver control, autonomous navigation, obstacle avoidance, and objects and robot interactions. Drag and drop blocks from the Robotics Playground Simulink library to create custom applications and use the getting started examples as a starting point for programming robot for different tasks.
Use the provided MATLAB Application in the "APPS" tab to easily navigate through the toolbox resources.
  • Open Robotics Playground App

System Requirements

This version of the toolbox was developed and tested in MATLAB 9.4 (R2018a).
